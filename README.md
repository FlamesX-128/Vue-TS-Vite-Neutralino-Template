# **Vue 3 + Typescript + Vite + Neutralino**
A template with Vue 3, TypeScript, Vite and Neutralinojs!


## **Starting**
Execute the following commands:

Yarn: ```$ yarn```

NPM: ```$ npm i```

```$ neu update```

### **Distribution**
Execute the following commands

Yarn: ```$ yarn build```
NPM: ```$ npm run build```
